# Visitors Counter

## Usage

### Install

#### Run on Repl.it

- Open the url [https://repl.it/@MarioYellowbear/Visitors-Counter](https://repl.it/@MarioYellowbear/Visitors-Counter)
- Just hit the **Fork** button
- And hit the **Run** button
